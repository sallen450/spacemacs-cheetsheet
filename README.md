# spacemacs-cheetsheet

## Term

* `M-m` 表示 `ESC m`

## Hotkey

### Buffer

* `SPC b d` 关闭当前 buffer
* `SPC b b` 列出所有 buffer
* `SPC b p/n` 上一个/下一个 buffer
* `SPC TAB` 切换到上一个 active 的 buffer

### Window

* `SPC w d` 关闭当前 window

### NeoTree (file tree)

* `j` `k` 上下移动
* `h` `l` 上一层或下一层
* `K` 返回到上一级目录

### Other

* `SPC s c` 清除 search 结果高亮
